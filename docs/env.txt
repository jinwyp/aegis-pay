
安装依赖包：
npm install
----------------------------------------

安装组件：
bower install
---------------------------------------

pdf-image转换文件需要配置：
ubuntu:
  sudo apt-get install imagemagick ghostscript poppler-utils
osx:
  brew install imagemagick ghostscript poppler
----------------------------------------------------------------------

项目启动命令：
  不用nock模拟api请求：
    npm start

  使用nock模式api请求：
    npm test
-----------------------------------------------------------------
