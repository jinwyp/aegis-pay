# aegis-pay
支付项目

## 安装依赖
backend 目录下：
- npm install
- brew install imagemagick ghostscript poppler    (OSX)
- sudo apt-get install imagemagick ghostscript poppler-utils    (Ubuntu)

app 目录下：
- npm install
- bower install

## 启动
- npm test  (使用nock)
- npm start (不使用nock)


## 访问
- http://localhost:8800
